Linux命令汇总

Linux系统

什么是WSL

WSL：Windows Subsystem for Linux，是用于Windows系统上的Linux子系统，作用很简单，可以在Windows系统中获得Linux系统环境，并且完全直连计算机硬件，无需通过虚拟机虚拟硬件。

功能：无需虚拟机，直接使用主机的硬件设备，构建linux操作系统。并不影响windows系统本身的运行

Linux命令基本格式

无论是上面命令，用于什么用途，在linux中，命令有其通用的格式：

command [-options] [parameter]

​ command：命令本身

​ -options：[可选]命令的一些选项，可以通过选项控制命令的行为细节

​ parameter：[可选]命令的参数，多数用于命令的指向目标等

ls命令

展示当前工作目录

可选项：-a，展示隐藏的，以.开头的文件

​ -l，以列表方式展示，

​ -h，需要和-l搭配，显示文件大小

cd命令

cd命令可以切换当前工作目录

​ cd 路径

​ 可以进入路径

​ cd /

​ 进入根目录

pwd命令

pwd命令用于显示当前工作目录

mkdir命令

mkdir用以创建新的文件夹

mkdir 路径

创建文件

-p可以创建不存在的父文件

touch命令

用于创建一个新的文件

touch 路径

参数必填，表示要创建的文件的路径，相对绝对，特殊路径都可使用

cat命令

用于查看文件内容

cat 路径

参数必填，表示要查看的文件的路径，相对绝对特殊路径都可以使用

more命令

用于查看文件内容，可翻页查看

more 路径

参数必填，表示要查看的文件的路径，相对绝对，特殊路径都可以使用

使用空格翻页，使用q退出

cp命令

用于复制文件或者文件夹

语法：cp [-r] 参数1，参数2

-r：可选，用于复制文件夹使用，递归文件夹

参数1：Linux路径，表示被复制的文件或文件夹

参数2：Linux路径，表示要复制去的地方

mv命令

用于查看文件内容

语法：mv 参数1，参数2

参数1：Linux路径，表示被移动的文件或者文件夹

参数2：Linux路径，表示要移动去的地方，如果目标不存在，则进行更名，确保目录存在

rm命令

用于复制文件或文件夹

语法：rm [-r -f]参数一 参数2 ··· 参数N

-r选项：可选，用于强制删除（）

-f选项：可选，用于强制删除

参数：表示被删除的文件或文件路径，支持多个，空格隔开

​ 可以使用通配符* ，模糊匹配

which命令

查找命令的程序文件

语法：which 要查找的命令

无需选项，只需要参数表示查找哪一个命令

find命令

用于查找指定的文件

按文件名查找：find 起始路径 -name “被查找文件名”

​ 支持通配符

按文件大小查找：find 起始路径 -size +|- n[kMG]

grep命令

从文件中查找并显示该文件行

语法：grep [-n] 查找字段 文件路径

选项-n 可选，表示在结果中显示匹配的行的行号

参数：查找字段，必填，表示要过滤的关键字，建议使用“”将关键字包围起来

参数：文件路径，必填，表示要过滤内容的文件路径，可作为管道符的输入

如果在管道符的后面，查找上一个命令的输出

wc命令

命令统计文件的行数，单词数量，字节数，字符数等

语法：wc[-c -m -l -w] 文件路径

不带选项默认统计全部

-c 字节数， -m字符数 ，-l行数 ，-w单词数

参数：被统计的文件路径，可作为管道符的输入

echo命令

可以使用echo命令在命令行中输出指定内容

语法：echo 输出的内容

tail命令

查看尾部内容，并持续跟踪

语法：tail [-f -num] Linux路径

查看尾部的num行 默认10

-f ：追踪更改

Linux路径，表示被查看的文件

ln命令

创建软连接

指向不在当前工作目录的文件或者文件夹并非物理移动，只是借助连接跳转到该文件

语法：ln -s 被连接的文件 目的地

-s 创建软连接

date命令的作用和用法

date命令可以查看日期时间并且格式化

语法：date 【-d】 格式化字符串

Y y（1999只显示99） m d H M S s（时间戳，1970-1-1到现在的秒数）

如何修改时区

rm -f /ete/localtime

sudo ln -s /usr/share/zoneinfo/Asia/Shanghai /etc/localtime

ntp的作用：可以自动联网同步时间，也可以通过ntpdate -u ntp.aliyun.com手动校准

ping命令

语法：ping -c num ip 或者主机名称

-c ： 测试次数，与num搭配使用

wget命令

可以进行网络文件下载

语法：wget -b（后台下载） url

curl命令

可以发起网络请求，类似于访问页面

语法：curl -o url

-o 用于下载

ps命令

ps -ef 查看进程信息

ps -ef | grep 关键字，过滤指定关键字进程信息

kill命令

kill 【-9】 关闭进程

-9：强制关闭进程

top命令

类似于windows任务管理器

选项
-p ：只显示某个进程的信息
-d：设置刷新时间，默认是5s
-c：显示产生进程的完整命令，默认是进程名
-n：指定刷新次数，比如 top -n 3，刷新输出3次后退出
-b：以非交互非全屏模式运行，以批次的方式执行top，一般配合-n指定输出几次统计信息，将输出重定向到指定文件，比如 top -b -n 3 > /tmp/top.tmp
-i：不显示任何闲置 (idle) 或无用 (zombie) 的进程

-u：查找特定用户启动的进程

交互式：

h键按下h键，会显示帮助画面
c键按下c键，会显示产生进程的完整命令，等同于-C参数，再次按下c键，变为默认显示
f键按下f键，可以选择需要展示的项目
M键按下M键，根据驻留内存大小(RES) 排序
P键按下P键，根据CPU使用百分比大小进行排序
T键按下T键，根据时间/累计时间进行排序
E键按下E键，切换顶部内存显示单位
e键按下e键，切换进程内存显示单位
1键按下1键，切换显示平均负载和启动时间信息。
i键按下i键，不显示闲置或无用的进程，等同于-i参数，再次按下，变为默认显示
t键按下t键，切换显示CPU状态信息
m键按下m键，切换显示内存信息

df命令

查看磁盘使用率

df -h

iostat命令

查看磁盘速率

sar -n Dev 命令

查看网络情况

rz、sz命令

先通过yum -y install lrzsz 可以安装此命令

rz进行文件上传

sz进行文件下载

Linux系统常用的压缩格式有:

tar格式，归档文件，简单的将文件整合到一个文件内，无压缩效果gzip格式，gzip压缩文件，不仅能整合到一个文件，同时有体积压缩效果

tar命令

tar[-z -X -V -c-f-C]参数
-c创建压缩文件、-V,查看压缩\解压过程、-X,解压模式-f,指定压缩\解压的文件，-z,gzip模式，-C,指定解压的路径
-z在选项组建议在开头，-f在选项组内必须在尾部，-C单独使用

zip命令

zip[-门] 参数…
-r,压缩文件夹使用

unzip命令

unzip 【-d】 参数

-d ：指定要压缩的目录

Linux用户管理

Linux支持多用户，多用户组，一个组多个用户

Linux权限管控的单元是用户级别和用户组级别

用户、用户组相关管理命令

groupadd 添加组

groupdel 删除组

useradd 添加用户

userdel 删除用户

usermod 修改用户组

id 查看用户信息，root下可指定查看

getent passwd 查看系统全部用户信息

getent group 查看系统全部组信息

su命令

可以切换用户

语法：su - 用户名

用户名省略默认切换到root

用户管理权限

ls -l列出的权限信息如何解读

第一列，表示文件、文件夹的权限控制信息
第三列，表示文件、文件夹所属用户
第四列，表示文件、文件夹所属用户组

rwx的意义

r：查看

w：修改

x：执行

chmod命令

功能：修改文件，文件夹的权限细节

限制：只能是文件、文件夹的所属用户或者root有权修改

语法：chmod 【-R】 权限 文件

选项：-R ，对文件夹内的全部内容应用同样规则

权限的数字序号：二进制表示，r代表4，w代表2，x代表1

​ rwx的相互组合可以得到0-7共8种组合

​ eg：rwx表示7 r-x表示5

chown命令

功能：修改文件文件夹的所属用户和用户组

限制：限root执行

语法：chown 【-R】 用户：用户组 文件

选项：-R 同chmod

sudo命令

可以让普通用户的命令具有root权限

语法：sudo 其他命令

前置条件，需要让root用户执行visudo命令，增加配置可让普通用户有sudo命令的执行权限

管道符

依次执行命令

将前一个命令的结果，作为下一个命令的输入

反引号符

被‘（飘号~）包围的内容，会被当作命令执行

重定向符

一个> 表示覆写

两个> 表示追加写入

vim编辑器

就是命令行模式下的文本编辑器，用来编辑文件

基础命令：vim 文档路径

运行模式：

​ 命令模式：默认的模式，可以通过键盘的快捷键控制文件内容

​ 输入模式：通过命令模式进入，可以进行编辑，esc返回命令模式

​ 底线命令模式：通过命令模式进入，可以对文件进行保存，关闭等 等

小技巧

ctrl+c 强制退出

ctrl+d 退出登出

history 查看历史命令

！命令前缀，自动匹配上一个命令

ctrl+r 搜索历史命令

ctrl+a|e 光标移动到头或尾

chrt+左右箭头，左右跳单词

ctrl + l 清屏

在linux中使用命令联网管理软件安装

yum -y install|remove|search 软件名称

Ubuntu系统：

apt -y install|remove|search 软件名称

systemctl命令

可以控制服务的启动关闭，开机自启动

​ 系统内置服务均可被systemctl控制

​ 第三方软件，如果自动注册了可以被systemctl控制

​ 第三方软件，如果没有自动注册，可以手动注册

语法：systemctl start |stop | status（查看状态是否启动）|enable|disable（关闭开机自启动） 服务名ez
